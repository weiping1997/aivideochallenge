# AI Video Challenge 2026 - Project Context

## Project Overview
Landing page for AI Video Challenge 2026 - a collaborative initiative between Jooy Media and Rtist to drive registrations for a FREE 3-hour AI Video Workshop and subsequent contest with RM3,800 in prizes.

## Key Objectives
- Achieve 100+ qualified registrations
- 70%+ conversion rate target
- Mobile-first responsive design
- Fast page load (<3 seconds)

## Core Features Required
1. **Hero Section** - Value proposition, CTA, co-organizer logos
2. **Workshop Details** - Online, 3 hours, FREE, limited 100 seats
3. **Contest Info** - 30s-1min video submission, "Upskill in 2026" theme
4. **Prizes** - 1st: RM2,000 + Advanced AI Class + Rtist Premium, 2nd: RM1,000 + AI Class + Rtist Plus, 3rd: RM800 + AI Class + Rtist Plus
5. **Registration Form** - Name, Email, Phone, Occupation dropdown, optional referral source
6. **FAQ Section** - Expandable accordion
7. **Timeline** - Workshop date TBC, submission deadline Feb 10, 2026

## Technical Stack
- Frontend: React.js/Next.js + Tailwind CSS
- Form: Google Forms/Typeform/n8n custom integration
- Hosting: Vercel/Netlify
- Analytics: GA4, event tracking
- Email: Automated sequences (confirmation, reminder, access link)

## Design Requirements
- Modern, AI-themed, professional
- Jooy Media + Rtist brand colors
- High-contrast CTAs (orange/bright blue)
- Mobile-first (60%+ mobile traffic expected)
- WCAG 2.1 AA accessibility
- Touch-friendly (44x44px minimum)

## Performance Targets
- LCP <2.5s, FCP <1.8s, CLS <0.1
- Google PageSpeed 90+
- WebP images, lazy loading
- CDN for static assets

## Success Metrics
- 100+ registrations
- 70%+ conversion rate
- <40% bounce rate
- <3% form abandonment
- 2+ min average time on page

## Timeline
15-20 days total development (Planning: 2d, Design: 3-4d, Dev: 5-7d, Content: 2-3d, Testing: 2-3d, Launch: 1d)

## Stakeholders
- Jooy Media: Content, workshop delivery, marketing
- Rtist: Community outreach, prizes, judging
- Developer: Landing page build, integration
- Designer: Visual design, brand compliance
